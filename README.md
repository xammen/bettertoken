# BetterToken

Token usage tracker for OpenCode TUI.

## Install

```bash
npm i -g opencode-bettertoken
```

## Use

Add this to `~/.opencode/tui.jsonc`:

```jsonc
{
  "plugin": ["opencode-bettertoken"]
}
```

## What it does

- Tracks token usage across sessions
- Shows usage in the sidebar footer
- Shows prompt stats in the session prompt area
- Supports configurable provider presets and token counting modes
